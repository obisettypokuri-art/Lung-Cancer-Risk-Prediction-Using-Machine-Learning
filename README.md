## ūüďĆ Overview

Lung cancer remains one of the leading causes of cancer-related mortality worldwide. Early detection and risk stratification can dramatically improve patient outcomes by enabling timely clinical intervention.

This case study applies a **full production-grade ML pipeline** ‚ÄĒ from raw data to clinically actionable risk scores ‚ÄĒ using 4,163 patient records across 14 features. The pipeline covers supervised classification, unsupervised clustering, hyperparameter tuning, model explainability, and a three-tier clinical risk scoring system.

**Models trained and compared:**

| Model | Type | Purpose |
|-------|------|---------|
| ūüĆ≤ Random Forest (baseline) | Ensemble | Primary classifier |
| ūüďČ Logistic Regression | Linear | Interpretable baseline |
| ūüĒß Random Forest (tuned) | Ensemble + GridSearchCV | Optimised classifier |
| ‚ö° XGBoost | Gradient Boosting | Industry-standard tabular ML |
| ūüéĮ SVM (RBF kernel) | Kernel method | High-dimensional margin classifier |
| ūüĒĶ K-Means Clustering | Unsupervised | Patient risk stratification |

## ūüď¶ Dataset

**Multi-cohort Lung Cancer Patient Dataset**

| Detail | Value |
|--------|-------|
| ūüĎ§ Records | 4,163 patient encounters |
| ūüß¨ Features | 14 (13 predictors + 1 target) |
| ūüéĮ Target | `lung_cancer` ‚ÄĒ binary (0 = Low risk, 1 = Medium‚ÄďHigh risk) |
| ‚öĖÔłŹ Class Balance | ~79% positive (imbalanced) ‚ÄĒ handled via stratified splits |
| ūüĒó Source | Multiple hospital CSV cohorts, concatenated |

### Feature Summary

| Feature | Type | Description |
|---------|------|-------------|
| `age` | Numeric | Patient age (range: 14‚Äď87) |
| `gender` | Encoded Integer | 0 / 1 / 2 |
| `smoking` | Integer | Smoking habit indicator |
| `alcohol` | Integer | Alcohol consumption |
| `air_pollution` | Float | Exposure level (imputed) |
| `occupational_hazards` | Float | Workplace hazard score (imputed) |
| `genetic_risk` | Float | Genetic predisposition (imputed) |
| `chest_pain` | Integer | Clinician-rated severity (1‚Äď9) |
| `cough` | Float | Clinician-rated severity (imputed) |
| `shortness_of_breath` | Integer | Clinician-rated severity |
| `fatigue` | Float | Clinician-rated severity (imputed) |
| `wheezing` | Integer | Wheezing severity |
| `swallowing_difficulty` | Float | Clinician-rated (imputed) |
| `lung_cancer` | Integer (Target) | 0 = No/Low ¬∑ 1 = Yes/Medium‚ÄďHigh |

> **Note:** Features marked *(imputed)* contain ~48‚Äď52% missing values originating from different cohort CSV files. All are filled using column mean before training.

## ūüĒć Model Explainability (SHAP)

SHAP (SHapley Additive exPlanations) is used to make the best model auditable by clinicians:

- **Beeswarm Plot** ‚ÄĒ shows the distribution of each feature's impact across all test patients
- **Bar Plot** ‚ÄĒ ranks features by mean absolute SHAP (global importance)
- **Force Plot** ‚ÄĒ explains a single high-risk patient's prediction feature by feature

> Top predictive features identified: `genetic_risk`, `smoking`, `air_pollution`, `chest_pain`, `shortness_of_breath`

## ūüß† Key Concepts

| Term | Definition |
|------|------------|
| **Stratified Split** | Train/test split preserving the 79/21 class ratio in both sets |
| **Mean Imputation** | Replacing missing values with the column mean ‚ÄĒ preserves scale |
| **StandardScaler** | Transforms features to zero mean, unit variance |
| **GridSearchCV** | Exhaustive hyperparameter search with cross-validation |
| **XGBoost** | Gradient boosting ‚ÄĒ trees built sequentially to correct prior errors |
| **scale_pos_weight** | XGBoost parameter to handle class imbalance (neg/pos ratio) |
| **ROC-AUC** | Measures class separability (1.0 = perfect, 0.5 = random) |
| **F1-Score** | Harmonic mean of Precision & Recall ‚ÄĒ ideal for imbalanced data |
| **SHAP** | Game-theory attribution of each feature's contribution per prediction |
| **Elbow Method** | Selects optimal k for K-Means by plotting inertia vs. cluster count |
